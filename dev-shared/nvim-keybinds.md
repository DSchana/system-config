# Neovim Keybinds

## Navigation (splits)

| Key   | Action             |
| ----- | ------------------ |
| `C-h` | Move cursor left   |
| `C-j` | Move cursor down   |
| `C-k` | Move cursor up     |
| `C-l` | Move cursor right  |

## Window Resize

| Key   | Action       |
| ----- | ------------ |
| `A-h` | Resize left  |
| `A-j` | Resize down  |
| `A-k` | Resize up    |
| `A-l` | Resize right |

## Telescope (find)

| Key           | Action                  |
| ------------- | ----------------------- |
| `<leader>ff`  | Find files              |
| `<leader>fr`  | Recent files            |
| `<leader>fs`  | Live grep (search)      |
| `<leader>fc`  | Grep string under cursor|
| `<leader>fb`  | Find buffers            |
| `<leader>fd`  | Find diagnostics        |

## Telescope (LSP)

| Key           | Action                  |
| ------------- | ----------------------- |
| `<leader>ld`  | LSP definitions         |
| `<leader>lr`  | LSP references          |
| `<leader>ls`  | LSP document symbols    |
| `<leader>lw`  | LSP workspace symbols   |

## LSP

| Key           | Action                  |
| ------------- | ----------------------- |
| `gd`          | Go to definition        |
| `gr`          | Go to references        |
| `gI`          | Go to implementation    |
| `K`           | Show hover (type/docs)  |
| `gl`          | Show diagnostic (error) |
| `<leader>ca`  | Code action             |
| `<leader>cs`  | Organize imports        |

## File Explorer

| Key           | Action                  |
| ------------- | ----------------------- |
| `<leader>ee`  | Toggle file explorer    |
| `<leader>ec`  | Collapse file explorer  |
| `<leader>er`  | Refresh file explorer   |

## Completion (insert mode)

| Key       | Action                        |
| --------- | ----------------------------- |
| `C-b`     | Scroll docs up                |
| `C-f`     | Scroll docs down              |
| `C-Space` | Trigger completion            |
| `C-e`     | Abort completion              |
| `Enter`   | Confirm selection             |
| `Tab`     | Next item / expand snippet    |
| `S-Tab`   | Previous item                 |
